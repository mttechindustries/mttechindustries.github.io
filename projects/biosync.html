BioSync

Privacy-First Chemistry Score Engine

A mobile biometric platform that computes a real-time Chemistry Score — a holistic physiological state metric — by fusing camera-based heart rate variability (rPPG) with on-device sentiment analysis. All processing runs locally. No biometric data leaves the device.

The Chemistry Score

Multi-Signal Physiological State Metric

The Chemistry Score is a proprietary composite metric designed to reflect an individual's real-time mental and physiological state without wearable hardware. It fuses three weighted data streams, all computed on-device.

HRV Score40% weight — Camera rPPG → RMSSD
Sentiment Score30% weight — On-device NLP (DistilBERT)
User Feedback30% weight — Contextual 1–5 rating

Core Technology

Camera-Based rPPG Heart Rate Variability

BioSync extracts cardiac data from standard phone cameras using remote photoplethysmography (rPPG) — detecting subtle color changes in facial skin caused by blood volume pulse. No contact required, no wearable hardware needed.

Green Channel Extraction

ROI-based green channel averaging isolates the signal most sensitive to blood volume pulse changes from the raw camera feed.

Butterworth Band-Pass Filter

5th-order filter isolates the physiological frequency range (0.7–4.0 Hz), cleaning the raw signal before peak detection.

R-Peak Detection

SciPy find_peaks identifies inter-beat intervals (IBI) in milliseconds — the foundation for RMSSD-based HRV score calculation.

Motion Artifact Correction

Accelerometer data flags high-motion events. Adaptive filtering using ICA/PCA removes motion artifacts before RMSSD computation.

On-Device Privacy Architecture

Every computation in BioSync runs locally. No biometric data, HRV values, sentiment scores, or Chemistry Scores are transmitted to external servers. The system is architecturally incapable of cloud-leaking sensitive physiological information.

Local Processing Stack

  • React Native (iOS/Android) for camera capture
  • Python/Flask backend running on-device
  • MobileBERT / TF.js for sentiment NLP
  • NumPy + SciPy signal processing pipeline
  • Firebase (auth only) — no biometric sync

Privacy Guarantees

  • Camera data never leaves device memory
  • NLP model weights stored locally
  • Chemistry Score computed and stored on-device
  • No cloud biometric storage or processing
  • User controls all data retention
