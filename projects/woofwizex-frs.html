Project Overview

WoofWizeX-FRS represents a breakthrough in pet emotion recognition technology, utilizing WiFi Channel State Information (CSI) to detect and interpret canine emotional states without requiring any wearable devices.

The platform combines advanced signal processing with machine learning algorithms to create a non-intrusive, privacy-preserving solution for pet owners and veterinarians.

WiFi-based Detection

Utilizes CSI from standard WiFi routers

Edge Deployment

ESP32-based edge computing solution

Multi-modal ML

Combines multiple data sources for accuracy
